Basic of c : 

Datatypes : A data type specifies the type of data that a variable can store such as integer, floating, character, etc.
Example:
int a=10;
float a=10.0;
char a='z';

Identifiers : A C Identifier is a name used to identify a variable, function, or any other user defined item. 
              An identifer starts with a letter A to Z, a to z, or an underscore '_' followed by zero or more letters, underscores, and digits (0 to 9).

Keywords : These reserved words (keywords) may not be used as constants or variables or any other identifier names.

auto          else          long
register      typedef       case
extern        return        union

goto          sizeof        volatile
default       if            static
while         do            int
switch        break         double
_packed       enum

Semicolons : In a C program, the semicolon is a statement terminator. That is, each individual statement much be ended with a semicolon. 
             It indicates the end of one logical entity.
